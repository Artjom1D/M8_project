#  CareerNavigator Bot

Умный Telegram-бот для поиска карьерных возможностей. Помогает подросткам и людям, ищущим смену профессии, найти вакансии, соответствующие их интересам и уровню подготовки.

**Целевая аудитория:** студенты, молодые специалисты, люди в переходе между карьерами.

---

##  Основные возможности:

###  Для пользователей:
- ** Поиск вакансий** — по ключевым словам, категориям, уровню опыта
- ** Персональные рекомендации** — на основе интересов и уровня пользователя
- ** Профиль** — сохранение интересов, уровня, истории поиска
- ** Информация о зарплатах** — диапазоны зарплат по должностям
- ** Справка** — встроенная помощь и инструкции

### Для администраторов:
- ** Добавление вакансий** — через бот или прямого импорта в БД
- ** Управление данными** — полный контроль над базой вакансий

##  Структура проекта:

```
M8_project/
├── bot.py              # Основной модуль Telegram-бота с обработчиками команд
├── logic.py            # Бизнес-логика: работа с БД, поиск, рекомендации
├── config.py           # Конфигурация (токен, пути)
├── bd.py               # Дополнительные утилиты для БД
├── jobs.db             # SQLite база данных (создаётся автоматически)
└── README.md           # Эта документация
```

###  Связи между модулями:
```
bot.py
  ├─ импортирует JobBot из logic.py
  ├─ использует config.py для параметров
  └─ управляет состояниями и UI

logic.py
  └─ работает с SQLite (jobs.db, users)
```

##  Структура базы данных:

### Таблица `jobs` (вакансии):
| Поле | Тип | Описание |
|------|-----|---------|
| id | INTEGER | Primary key, автоинкремент |
| company | TEXT | Название компании |
| title | TEXT | Должность |
| salary_from | INTEGER | Нижний предел зарплаты |
| salary_to | INTEGER | Верхний предел зарплаты |
| skills | TEXT | Требуемые навыки (через запятую) |
| level | TEXT | junior/middle/senior |
| category | TEXT | Категория (IT, Дизайн, Маркетинг, Наука, Бизнес) |

### Таблица `users` (профили):
| Поле | Тип | Описание |
|------|-----|---------|
| user_id | INTEGER | Telegram ID пользователя |
| name | TEXT | Имя пользователя |
| interests | TEXT | Интересы (через запятую) |
| level | TEXT | Уровень опыта |
| meta | TEXT | Доп. информация (резерв) |

##  Быстрый старт:

###  Установка зависимостей:
```bash
pip install pyTelegramBotAPI
```

###  Настройка конфига:
Убедись, что в `config.py` указан валидный токен Telegram:
```python
TOKEN = "твой_telegram_bot_token"
DB_PATH = "jobs.db"
```

###  Запуск бота:
```bash
python bot.py
```

**Ожидаемый вывод:**
```
Бот запущен 
```

###  В Telegram:
- Найди бота по имени (через @)
- Нажми `/start`
- Выбери нужное действие из меню

##  Примеры использования:

### Сценарий 1: Новый пользователь ищет вакансии
```
1. /start
2. Нажимает " Поиск"
3. Выбирает категорию "IT" или вводит "Python"
4. Получает 5 релевантных вакансий с деталями
```

### Сценарий 2: Сохранение профиля
```
1. "Профиль"
2. Вводит интересы: "python, веб-разработка, стартапы"
3. Вводит уровень: "junior"
4. При следующем поиске получит персональные рекомендации
```

### Сценарий 3: Добавление вакансии (админ)
```
1. " Добавить вакансию"
2. Вводит: Google|Senior Python Developer|100000|150000|python,django,postgresql|senior|IT
3. Вакансия добавлена в БД 
```

## Формат импорта вакансий:

### Через бот:
```
company|title|salary_from|salary_to|skills|level|category
```

### Примеры:
```
Google|Junior Python Developer|40000|80000|python,sql,django|junior|IT
Apple|UX Designer|60000|120000|figma,ui,prototyping|middle|Дизайн
McKinsey|Business Analyst|50000|100000|analytics,excel,powerbi|junior|Бизнес
```

### Через SQL:
```sql
INSERT INTO jobs VALUES 
(NULL, 'Tesla', 'Full Stack Developer', 80000, 150000, 'react,node,postgres', 'middle', 'IT');
```
