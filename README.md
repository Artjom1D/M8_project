CareerNavigator Bot

## Инструменты для реализации:

Telegram Bot API — платформа для работы бота 
Язык программирования: Python
Библиотека для бота: python-telegram-bot
База данных: SQLite 
OpenAI API (для интеллектуальной адаптации ответов)

## Проект состоит из:

Удобный интерфейс
ИИ-ассистент, который сможет предложить вам работу взависемости от вашего запроса
Ручной поиск по базе данных с работами для пользавотеля

SS.com
Сv.lv
